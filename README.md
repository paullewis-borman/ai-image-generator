# ai-image-generator

Self-contained tool for updating an image via the OpenRouter API. Everything lives in [`ai-image-update/`](ai-image-update/README.md) — see that folder's README for setup and usage.

To use this in another project: copy the `ai-image-update/` folder into it as-is. It resolves its own paths and carries its own `.env`, `.gitignore`, prompts, and assets, so no changes to the host project are needed.
