# Generic image instructions

## Output format

- Aspect ratio: 16:9
- Dimensions: 2752x1536 (2K tier at 16:9 — for a website hero image)
- File format: PNG
- Style: Photorealistic

## What may be changed

- The character / person in the foreground of the image
- The clothes, appearance or pose of the character as long as they do not conflict with the items that may not be changed
- The tools in the persons toolbelt can be changed

## What may NOT be changed

- Any part of the background of the image - setting, lighting etc.
- The character location - the character MUST remain within the same portion of the image
- The output format specified may not be changed
- The logo on the T-shirt must say "Bence" then underneath "Builders Merchants"
- the character MUST clearly be a tradesperson
- the character should have a friendly confident appearance and be smiling at the camera
- Any part or element of the image not explicitly specified in the combined prompt

## Precedence

These rules override any conflicting instruction from the user's prompt. If a
user request conflicts with a rule above, follow this file and ignore that
part of the request.
