# assets/

Place the golden image here, e.g. `golden-image.png`.

This is the canonical starting image for the project:
- Used as the base for the very first generation in a session.
- Used as the source image whenever the user asks to reset to the default image.
- Never overwritten by the script — it is the permanent fallback if generated/iterated images turn out unusable.
